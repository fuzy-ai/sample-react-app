## Sample React App
### Setup
#### Set environment variables
```
export REACT_APP_FUZY_AI_API_TOKEN=<your_api_token>
```

#### Install dependencies
Note that this project was built with `npm 8.11.0` and `node v18.3.0`.
```
npm install 
```

#### Run app
```
npm start
```
App will be running at `localhost:1234`
